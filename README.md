# Task Manager App

A simple application that allows users to create and manage tasks.

---

## 🚀 Features

- Add new tasks  
- Display tasks in a table  
- Simple and clean UI  

---

## 🛠 Tech Stack

- Frontend: React  
- Backend: Sprint boot
- Database: Postgres

---
